# Repository visual assets

## `repo-intro-gtvn-idl.png`

Opening graphic derived deterministically from a representative GTVn row of the manuscript selected-case figure (`fig_selected_case_gtvn_slice_overlays.png`). It retains the original medical-image pixels and contour overlays and changes only crop, scale, spacing, panel labels and legend placement.

- Views: sagittal and axial.
- Background modality: CT, as used by the selected-case GTVn generation script.
- Comparison: multi-observer reference labels versus corresponding post-iDL predictions.
- Identity handling: no case name, patient identifier or internal ID is displayed.
