# Repository visual assets

## `repo-intro-gtvt-idl.png`

Opening graphic derived deterministically from a representative GTVt row of manuscript Figure 5 (`fig_selected_case_gtvt_slice_overlays.png`). It retains the original medical-image pixels and contour overlays and changes only crop, scale, spacing, headings and legend placement.

- Views: sagittal and axial.
- Background modality: CT, as used by the Figure 5 generation script.
- Comparison: multi-observer reference annotations versus corresponding post-iDL predictions.
- Identity handling: no case name, patient identifier or internal ID is displayed.
